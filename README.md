# daily-blog-frontend
A clean and responsive daily blog website UI built using HTML and CSS, focusing on layout design and modern styling.
This project focuses on creating a visually appealing layout for daily blog content with modern styling and simple structure.

Features:
Responsive design (mobile-friendly)
Clean and structured layout
Modern UI with CSS styling
Blog post cards / sections
Easy to customize and extend

Tech Stack:
HTML5
CSS3

How to Run

Clone the repository:
git clone https://github.com/eyekaushal/daily-blog-frontend.git
Open index.html in your browser.
